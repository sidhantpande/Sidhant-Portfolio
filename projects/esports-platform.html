Haunt the Arena

HTML5 • CSS3 • JavaScript • Three.js • Firebase • Firestore Auth

A production-ready esports tournament website for the "Haunt the Arena" Halloween gaming event, securing 150+ team registrations. Provides a complete tournament management experience — from player/team registration through bracket generation to winner selection — wrapped in a spooky, high-performance UI with 3D particle effects and immersive animations.

Key Features

  • Halloween-themed UI with glitch text, neon glows, fog effects
  • Three.js WebGL-powered 3D floating particle background
  • Tekken 7 (1v1) and BGMI Arena (4v4 TDM) tournament brackets
  • Multi-step animated registration wizard with real-time validation
  • Interactive single-elimination bracket viewer with live updates
  • Authenticated admin panel for managing registrations & winners
  • Contextual audio system with mute toggle
  • Firebase backend — Firestore, Auth, Hosting ready
