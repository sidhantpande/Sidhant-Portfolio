## ✨ The Edge

This portfolio isn't just a list of skills—it's a demonstration of technical proficiency and elite UI/UX design.

*   **Fluid Liquid Backgrounds**: Features a custom-built 20-second animated `@keyframes` mesh radial gradient background.
*   **The Frosted Glass Engine**: All content cards across "Experience", "Skills", and "About" utilize a deeply refined `backdrop-filter: blur(20px)` geometry, simulating heavy frosted glass levitating above the animated background.
*   **Sticky Rail Architecture**: Professional Experience is rendered through an expansive, horizontal sticky-rail Grid Flow, maximizing readability and metric-driven storytelling.
*   **Lenis Smooth Scrolling**: Overhauled internal mechanics with proper nested wheel normalization (`smoothWheel: true`) and strict iframe scroll boundary blocking to ensure perfect 60FPS inertial scrolling.
*   **Precision Typography**: High-impact contrast between `Outfit` for massive headers and `Inter` for hyper-legible body text.
