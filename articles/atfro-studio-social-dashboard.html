PRODUCT DEEP DIVE

Atfro Studio: High-Performance Social Media Dashboard.

A heavily optimized, high-performance Next.js fullstack SaaS built explicitly for YouTube content creators.

The Vision

Atfro Studio is a flagship product of ATFRO, the transformation venture started by Sidhant Pande. The application radically simplifies the complex Google YouTube API Data v3 payload, delivering flawless, low-latency insights and storytelling analytics.

Built explicitly for professional creators, it utilizes a custom Postgres Cache layer and robust Clerk authentication to provide a seamless experience.

The Engineering Handbook

The system was built under strict architectural rules constructed for production deployment. The core implementation focuses on a monolithic boundary managed through detailed internal documentation.

Core Systems

  • Architecture Breakdown: Layered system design with UI, API handlers, and services.
  • Backend Systems: Next.js App Router API routes with service-role Supabase clients.
  • Frontend Hierarchy: Composed React components powered by custom SWR hooks.
  • Caching Strategy: Multi-tier TTL-driven cache utilizing Supabase analytics_cache.
  • Security Core: 32-Byte hex mapping for OAuth encryption and token persistence.

Technology Stack

Leveraging a modern, battle-tested stack for performance and developer velocity:

  • Framework: Next.js 16 (App Router) + React 19.
  • Identity: Clerk Authentication wrapping Server bounds.
  • Database: PostgreSQL managed via Supabase.
  • Styling: Tailwind CSS V4 with Radix UI and Shadcn primitives.
  • Data Fetching: SWR for client-side deduping and revalidation.

Environment & Deployment

The application follows a rigorous environment initialization process, requiring exactly 32-Byte hex mapping for token encryption to ensure maximum security for creator data.
